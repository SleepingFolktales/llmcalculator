# binaries/

This directory holds the PyInstaller-built Python backend binary.

The file is generated by the build script and must be named with the Rust
target triple so Tauri can bundle the correct binary for each platform:

| Platform | Filename |
|---|---|
| Windows x64 | `llmcalc-backend-x86_64-pc-windows-msvc.exe` |
| macOS Apple Silicon | `llmcalc-backend-aarch64-apple-darwin` |
| macOS Intel | `llmcalc-backend-x86_64-apple-darwin` |
| Linux x64 | `llmcalc-backend-x86_64-unknown-linux-gnu` |

## Generate the binary

From the `llmcalculator/` directory:

**Windows:**
```bat
scripts\build-backend.bat
```

**macOS / Linux:**
```bash
bash scripts/build-backend.sh
```

The script detects your target triple automatically and places the binary here.
